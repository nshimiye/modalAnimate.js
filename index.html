animatedModal.js

animatedModal.js is a jQuery plugin to create a fullscreen modal with CSS3 transitions. You can use the transitions from odalanimate.css or create your own transitions.

Options

Variable Default Value Options Description
modalTarget modal-container Customize your modal target
effect unfolding 'unfolding', 'revealing', 'uncovering', 'blown-up', 'meep', 'sketch', 'bond' Effects for modal
autoEffect false true / false (Boolean) Effects for multiple listeners Here
